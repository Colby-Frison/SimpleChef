The project will follow the course milestones and an internal task breakdown:

\begin{itemize}
  \item \textbf{Project Proposal (Current)}: Define scope, complete proposal document with wireframe, and finalize design specifications.
  \item \textbf{Weeks 1--4 (Through March 26)}: Core prototype development. Implement recipe library, step-by-step cooking mode with timer dock, and meal planner calendar. Conduct initial user feedback. Deliver Progress Report with GANTT chart, prototype demo, and documentation.
  \item \textbf{Weeks 5--7 (March 27--April 28)}: Complete grocery list automation, recipe input flow (manual and AI parsing), profile and settings. User testing and iteration. Finalize functional prototype. Project Delivery due April 28.
  \item \textbf{Final Demo (Starting April 9)}: In-class presentation with live demonstration of the prototype, summary of design decisions, and UX improvements.
  \item \textbf{April 28--May 3}: Final documentation, system block diagram, design rationale, and user analysis results. Final Project Report due May 3.
\end{itemize}
