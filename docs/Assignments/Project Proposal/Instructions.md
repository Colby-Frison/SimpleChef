Overview: This final project focuses on Human-Computer Interaction (HCI) and user interface design. Working in teams, you will either analyze and improve an existing user interface or develop a new UI design concept. The project comprises a functional prototype that enhances user experience, along with proper documentation and a final presentation. The objective of this project is to provide hands-on experience in UI analysis, redesign, and implementation of HCI principles and to develop collaboration and technical skills.

Getting Started: Begin by either selecting an existing website or app interface with identifiable usability issues, or by developing a new UI design idea. Collaborate with your team to evaluate its strengths, weaknesses, and opportunities for improvement. Conduct a thorough analysis of user needs and challenges, documenting all identified UI issues and brainstorming potential solutions.

Proposal Requirements: Your proposal should include:

a) a project title and introduction that explains your chosen UI or new design concept;

b) a detailed section on issues with the existing UI (or problem statement for new designs);

c) clearly defined project goals that specify what you aim to improve or achieve;

d) a description of your proposed UI including key features and the tools you plan to use;

e) a planned timeline for project completion; and 

f) a simple wireframe, diagram, or mockup showing what your proposed UI or main dashboard will look like.

Project Evaluation Criteria

Your project will be evaluated based on the depth of your user goals analysis and how well you understand and address user needs. We will assess the creativity and justification of your design alternatives, as well as the quality of your prototype including its usability, functionality, responsiveness, and aesthetic appeal. Effective team collaboration with clear roles and contributions is essential.
